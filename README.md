# HR Analytics Dashboard (Power BI)

## Project Overview
This project focuses on analyzing employee data to identify workforce trends, attrition patterns, employee demographics, job satisfaction, compensation insights, and departmental performance. The dashboard helps HR teams make data-driven decisions to improve employee retention and workforce planning.

## Dataset Information
- Total Records: 1480
- Total Columns: 38
- Attrition Cases: 238
- Attrition Rate: 16.08%

## Business Problem
Employee attrition is a major challenge for organizations because it increases hiring costs, training expenses, and productivity loss. The goal of this project is to:
- Identify factors contributing to employee attrition.
- Analyze employee demographics and workforce distribution.
- Evaluate job satisfaction and work-life balance.
- Discover high-risk departments and employee groups.
- Support HR decision-making through data visualization.
